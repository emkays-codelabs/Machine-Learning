# 1.3 Types of Machine Learning

Machine Learning (ML) is a subfield of **Artificial Intelligence (AI)** that focuses on building algorithms and models that enable computers to **learn patterns from data and improve performance with experience**, without needing explicit programming for every task.

In simple terms, ML teaches systems to **analyze data, discover patterns and make decisions similar to humans**.

There are several types of machine learning, each designed for different kinds of problems.

The main types are:

```
1. Supervised Learning
2. Unsupervised Learning
3. Reinforcement Learning
```

Additionally, modern ML also includes:

```
4. Semi-Supervised Learning
5. Self-Supervised Learning
```

---

# 1. Supervised Machine Learning

## Definition

Supervised learning is a type of machine learning where the model is trained using a **labeled dataset**.

A labeled dataset contains:

```
Input Data (Features) + Correct Output (Label)
```

The algorithm learns a mapping function between inputs and outputs.

```
X → Y
```

Where:

```
X = Input features
Y = Target output
```

---

## Example

Suppose we train a model with labeled images:

| Image     | Label |
| --------- | ----- |
| Cat Image | Cat   |
| Dog Image | Dog   |

After training, when a new image is given, the model predicts:

```
Cat or Dog
```

---

## Categories of Supervised Learning

Supervised learning is mainly divided into two types:

```
1. Classification
2. Regression
```

---

# 1.1 Classification

Classification algorithms predict **categorical outputs**.

This means the output belongs to a specific class.

Example:

```
Spam / Not Spam
Disease / No Disease
Fraud / Legitimate
```

Example dataset:

| Email Text      | Label    |
| --------------- | -------- |
| Buy now         | Spam     |
| Meeting at 5 PM | Not Spam |

---

## Classification Algorithms

Common algorithms include:

```
Logistic Regression
Decision Tree
Random Forest
K-Nearest Neighbors (KNN)
Naive Bayes
Support Vector Machine (SVM)
```

---

# 1.2 Regression

Regression algorithms predict **continuous numerical values**.

Example problems:

```
House price prediction
Stock price prediction
Temperature forecasting
Sales forecasting
```

Example dataset:

| House Size | Price  |
| ---------- | ------ |
| 1000 sq ft | 200000 |
| 1500 sq ft | 300000 |

The model learns:

```
Price = f(Size)
```

---

## Regression Algorithms

```
Linear Regression
Polynomial Regression
Ridge Regression
Lasso Regression
Decision Tree
Random Forest
```

---

## Where to Use Supervised Learning

Supervised learning is used when:

* Data has **known labels**
* You want to **predict outcomes**

Typical tasks:

```
Classification
Regression
```

# 2. Unsupervised Machine Learning

## Definition

Unsupervised learning works with **unlabeled data**.

This means the dataset does not contain predefined outputs.

The algorithm automatically finds **patterns, relationships or structures in data**.

Example dataset:

| Customer | Spending |
| -------- | -------- |
| A        | 100      |
| B        | 120      |
| C        | 900      |
| D        | 850      |

The algorithm may group customers into:

```
Low spenders
High spenders
```

---

## Example

Customer segmentation.

Given customer purchase data, the model groups customers with similar behaviors.

This helps businesses target marketing campaigns.

---

# Categories of Unsupervised Learning

Main techniques include:

```
1. Clustering
2. Dimensionality Reduction
3. Association Rule Learning
```

---

# 2.1 Clustering

Clustering groups similar data points together.

Example:

Customer segmentation.

```
Cluster 1 → Budget customers
Cluster 2 → Premium customers
```

---

## Clustering Algorithms

```
K-Means
DBSCAN
Mean Shift
Hierarchical Clustering
```

---

# 2.2 Dimensionality Reduction

Large datasets may contain many features.

Dimensionality reduction reduces the number of features while preserving important information.

Example:

Reducing 100 variables to 10 important features.

---

## Dimensionality Reduction Techniques

```
Principal Component Analysis (PCA)
Independent Component Analysis (ICA)
t-SNE
```

---

# 2.3 Association Rule Learning

Association rule learning discovers relationships between items in datasets.

Example:

Market basket analysis.

```
Customers who buy bread often buy butter.
```

---

## Algorithms

```
Apriori
FP-Growth
Eclat
```

---

# Where to Use Unsupervised Learning

Unsupervised learning is useful when:

```
Data has no labels
Patterns need to be discovered
Exploratory data analysis is required
```

## Applications of Unsupervised Learning

Common applications include:

### Clustering and Segmentation

### Anomaly Detection


### Dimensionality Reduction

Used for:


### Recommendation Systems

Example:


# 3. Reinforcement Learning

## Definition

Reinforcement Learning (RL) trains an **agent** to make decisions through **trial and error**.

The agent interacts with an environment and receives feedback.

```
Correct Action → Reward
Wrong Action → Penalty
```

The goal is to learn the **best sequence of actions to maximize rewards**.

---

## Reinforcement Learning Components

```
Agent → Learner
Environment → World where agent operates
Action → Decision taken
Reward → Feedback signal
```

---

## Example

Chess playing AI.

The agent tries moves.

```
Good move → Positive reward
Bad move → Negative reward
```

Over time the model learns strategies to win.

---

## Reinforcement Learning Algorithms

```
Q-Learning
SARSA
Deep Q-Learning
Policy Gradient Methods
```

---

# Types of Reinforcement Learning

## Positive Reinforcement

Rewards desired behavior.

Example:

```
Points given for correct answers
```

---

## Negative Reinforcement

Removes negative outcomes to encourage good behavior.

Example:

```
Alarm stops when correct action is taken
```

# 4. Semi-Supervised Learning

Semi-supervised learning combines:

```
Supervised Learning
+
Unsupervised Learning
```

It uses:

```
Small labeled dataset
Large unlabeled dataset
```

This is useful when labeling data is **expensive or time-consuming**.

---

## Example

Language translation.

Instead of labeling millions of sentence pairs manually, the model learns from:

```
Few labeled translations
Large unlabeled text data
```

---

## Techniques

```
Graph-based Learning
Label Propagation
Co-training
Self-training
Generative Adversarial Networks (GANs)
```

---

## Applications

```
Image classification
Natural language processing
Speech recognition
Recommendation systems
Medical imaging
```

---

# 5. Self-Supervised Learning

Self-supervised learning is a modern technique where **models generate their own labels from raw data**.

The model learns by predicting parts of data from other parts.

Example:

In NLP models like BERT or GPT learn by:

```
Predicting missing words in sentences
```

Example sentence:

```
The sky is ___
```

Model predicts:

```
Blue
```

This helps models understand language without manual labeling.

---

## Techniques

```
Masked Language Modeling (BERT)
Contrastive Learning (SimCLR, MoCo)
Autoencoders
Predictive Coding
```

---

## Applications

Self-supervised learning is used in:

```
Natural Language Processing
Computer Vision
Speech Recognition
Video Understanding
Large AI Model Pretraining
```

# Comparison of Machine Learning Types

| Type            | Data Requirement | Label Availability    | Learning Goal          | Common Use Case        |
| --------------- | ---------------- | --------------------- | ---------------------- | ---------------------- |
| Supervised      | High             | Labeled               | Predict outputs        | Spam detection         |
| Unsupervised    | Medium           | Unlabeled             | Find hidden patterns   | Customer segmentation  |
| Reinforcement   | High             | Reward feedback       | Learn best actions     | Robotics, games        |
| Semi-Supervised | Medium           | Partial labels        | Combine learning types | NLP, image recognition |
| Self-Supervised | High             | Self-generated labels | Learn representations  | GPT, BERT              |

Machine Learning algorithms can be categorized based on how they learn from data.

Main types include:


Modern ML also includes:

```
Semi-Supervised Learning
```
